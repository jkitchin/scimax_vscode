* ‚úÖ Introduction
CLOSED: [2026-01-15 Thu 13:34]

The Agenda system in Scimax VS Code provides powerful views of your scheduled
tasks, deadlines, and TODO items. It helps you stay organized by showing what
needs to be done and when.

Think of the Agenda as your command center for task management - it collects
tasks from all your org files and presents them in meaningful ways:

- *Day/Week views* - See what's scheduled for specific time periods
- *TODO lists* - Review all incomplete tasks
- *Deadline views* - Track upcoming and overdue deadlines
- *Filtering* - Focus on specific tags, priorities, or categories

The Agenda system works with both file-based scanning and database indexing,
giving you flexibility for different workflows.

* ‚úÖ What is the Agenda?
CLOSED: [2026-01-15 Thu 13:34]

The Agenda is a dynamic view that displays TODO items, scheduled tasks, and
deadlines from your org files. Unlike browsing individual files, the Agenda
aggregates information across all your documents.

** ‚úÖ Key Concepts
CLOSED: [2026-01-15 Thu 13:34]

*** ‚úÖ Agenda Items
CLOSED: [2026-01-15 Thu 13:34]

An agenda item is any headline with:
- A TODO keyword (TODO, DONE, NEXT, WAITING, etc.)
- A SCHEDULED timestamp
- A DEADLINE timestamp
- An active timestamp

*** ‚úÖ Scheduled Items
CLOSED: [2026-01-15 Thu 13:34]

Tasks with SCHEDULED timestamps appear in the agenda on their scheduled date:

#+BEGIN_SRC org
* TODO Write report
SCHEDULED: <2026-01-15 Wed>
#+END_SRC

This appears in the agenda starting January 15.

*** ‚úÖ Deadlines
CLOSED: [2026-01-15 Thu 13:35]

Tasks with DEADLINE timestamps appear with warning indicators:

#+BEGIN_SRC org
* TODO Submit proposal
DEADLINE: <2026-01-20 Mon>
#+END_SRC

Shows in the agenda with countdown (e.g., "in 7 days") and becomes overdue if
not completed.

*** ‚úÖ Overdue Items
CLOSED: [2026-01-15 Thu 13:35]

Items past their deadline appear prominently with warning indicators and show
how many days overdue they are.

*** ‚úÖ Unscheduled TODOs
CLOSED: [2026-01-15 Thu 13:35]

TODO items without dates can be included in agenda views to ensure nothing is
forgotten.

** ‚úÖ Two Agenda Systems
CLOSED: [2026-01-15 Thu 13:35]

Scimax VS Code provides two complementary agenda implementations:

*** ‚úÖ Native/File-Based Agenda
CLOSED: [2026-01-15 Thu 13:35]

- Scans org files on-demand
- Works immediately without indexing
- Provides tree view in sidebar
- Commands: scimax.agenda.*
- Best for: Quick views, smaller document sets

*** ‚úÖ Database-Backed Agenda
CLOSED: [2026-01-15 Thu 13:35]

- Uses indexed data for fast queries
- Requires indexing files first
- Integrates with search and filtering
- Commands: scimax.db.agenda, scimax.db.deadlines
- Best for: Large document collections, complex queries

Both systems can be used together - choose based on your needs.

** ‚úÖ Agenda Item Data
CLOSED: [2026-01-17 Sat 12:34]

Database agenda items include:

#+BEGIN_SRC typescript
interface AgendaItem {
  type: 'deadline' | 'scheduled' | 'todo';
  heading: HeadingRecord;
  date?: string;
  days_until?: number;

#+END_SRC

Properties:

* ‚ö†Ô∏è Troubleshooting

** ‚ö†Ô∏è Agenda Shows No Items

*Issue*: Agenda view is empty or shows fewer items than expected.

*Solutions*:

1. *Check agenda files configuration*:
   #+BEGIN_SRC text
   Command: scimax.agenda.configure
   Ensure your org files are included
   #+END_SRC

2. *Verify TODO keywords*:
   - Check headlines have TODO, SCHEDULED, or DEADLINE
   - Ensure correct keyword spelling

3. *Check time range*:
   - Day view only shows today's items
   - Try week or month view for more items

4. *Reindex database* (for database agenda):
   #+BEGIN_SRC text
   Command: scimax.db.reindex
   #+END_SRC

5. *Check exclude patterns*:
   - Ensure files aren't excluded
   - Review scimax.agenda.excludePatterns

** ‚ö†Ô∏è Items Not Updating

*Issue*: Changed items don't appear or update in agenda.

*Solutions*:

1. *Refresh agenda*:
   #+BEGIN_SRC text
   Command: scimax.agenda.refresh
   #+END_SRC

2. *Reindex database*:
   #+BEGIN_SRC text
   Command: scimax.db.reindex
   #+END_SRC

3. *Check file watcher*:
   - Extension should auto-detect changes
   - Manual refresh if needed

4. *Save file*:
   - Ensure file is saved (C-s)
   - Agenda reads from disk

** ‚ö†Ô∏è Tree View Not Showing

*Issue*: Agenda tree view not visible in sidebar.

*Solutions*:

1. *Check view visibility*:
   - View ‚Üí Open View ‚Üí Scimax: Agenda

2. *Refresh view*:
   #+BEGIN_SRC text
   Command: scimax.agenda.refresh
   #+END_SRC

3. *Check for errors*:
   - View extension logs
   - Check for parsing errors in org files

** ‚ö†Ô∏è Slow Performance

*Issue*: Agenda is slow to load or refresh.

*Solutions*:

1. *Use database agenda*:
   - scimax.db.agenda is faster for large collections
   - Index files once, query quickly

2. *Limit agenda files*:
   - Don't include entire filesystem
   - Use specific directories

3. *Exclude large directories*:
   #+BEGIN_SRC json
   {
     "scimax.agenda.excludePatterns": [
       "**/node_modules/**",
       "**/vendor/**",
       "**/build/**"
     ]
   }
   #+END_SRC

4. *Optimize file sizes*:
   - Archive old DONE items
   - Split large files into smaller ones

** ‚ö†Ô∏è Date Display Issues

*Issue*: Dates show incorrectly or items appear on wrong days.

*Solutions*:

1. *Check timestamp format*:
   #+BEGIN_SRC org
   ; Correct
   SCHEDULED: <2026-01-15 Wed>

   ; Incorrect
   SCHEDULED: <01/15/2026>
   #+END_SRC

2. *Verify day of week*:
   - Day must match date
   - Use S-<right> to recalculate

3. *Check repeaters*:
   #+BEGIN_SRC org
   ; Ensure repeater syntax is correct
   SCHEDULED: <2026-01-15 Wed +1w>
   #+END_SRC
